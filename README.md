# ClickDown

**Google Chrome Extension**

一键获取当前页面上的所有`pdf`文件,打包成`zip`下载

# Libraries
[JSZip](http://stuartk.com/jszip)

[FileSaver.js](https://github.com/eligrey/FileSaver.js)
